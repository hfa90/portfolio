# 🎰 Lottery Statistical Analyzer — v2.0

Uma aplicação desktop robusta desenvolvida em Python para análise probabilística e estatística de resultados de loterias. Utiliza ferramentas avançadas de ciência de dados para gerar insights, gráficos interativos e previsões baseadas em modelos matemáticos (como Distribuição de Poisson, Teste Qui-Quadrado e Cadeias de Markov).

---

## ✨ Funcionalidades Principais

- **📊 Visão Geral:** Resumo estatístico rápido (médias, números mais/menos frequentes, soma, etc).
- **📈 Análise de Frequências:** Tabelas detalhadas de frequência absoluta/relativa e mapas de atraso de cada número.
- **🗺️ Gráficos Interativos:** - Heatmap (Mapa de calor) de dezenas.
  - Histogramas de distribuição.
  - Matriz de Transição de Markov.
  - Análise de pares vs ímpares e dispersão da soma.
- **🎲 Motor de Predição:** Geração de jogos otimizados baseados em diferentes estratégias (Balanceada, Números Quentes, Frios, Atraso ou Poisson).
- **🔬 Análise Avançada:** Testes de hipótese (Qui-Quadrado), Boxplots, Q-Q Plots e médias móveis ao longo do tempo.

---

## 🚀 Como Instalar e Rodar (Passo a Passo)

Recomendamos rodar o projeto dentro de um ambiente virtual para não causar conflitos com outras bibliotecas no seu computador.

### Passo 1: Preparar os arquivos
Certifique-se de que os seguintes arquivos estejam na mesma pasta:
1. `Interface Probablistica.py` (O código principal)
2. `loteria_sample.csv` (Arquivo de demonstração com resultados de loteria)

Crie um arquivo chamado `requirements.txt` nesta mesma pasta e cole o seguinte conteúdo dentro dele:
```text
pandas>=2.0.0
numpy>=1.24.0
scipy>=1.10.0
matplotlib>=3.7.0
